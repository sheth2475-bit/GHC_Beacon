Update Performo AI pricing, billing, activation, and platform owner controls.

Performo AI must support a platform-level owner role who controls company activation, license keys, subscriptions, and system-wide usage analytics.

====================================================
1. PLATFORM OWNER ROLE
====================================================

Add a top-level role called:
- Platform Owner

This role is above all company users and should have full control over the entire application platform.

Platform Owner should be able to:
- create and manage company accounts
- approve or reject activation requests
- generate activation keys / license keys
- assign plans to companies
- activate or deactivate company accounts
- extend trial periods if needed
- suspend accounts
- view all companies
- view overall user activity and usage statistics
- view AI usage statistics
- view login activity
- view company-level subscription and billing status
- monitor which users are actively using the application
- see feature usage trends
- see plan utilization and user limits

====================================================
2. ACTIVATION / LICENSE KEY SYSTEM
====================================================

Add an activation system for companies.

Requirements:
1. Each company should require activation before full use
2. Platform Owner can generate a unique activation key / license key
3. Platform Owner can send or assign the key to the company
4. Company Super Admin can enter the activation key to activate the application
5. Activation key should be linked to:
   - company
   - plan
   - status
   - expiry date if needed
   - max users
   - AI usage limit
6. Platform Owner must be able to:
   - create activation key
   - regenerate key
   - revoke key
   - extend key validity
   - mark company as activated manually if needed

Activation statuses:
- Pending Activation
- Active
- Trial Active
- Expired
- Suspended
- Deactivated

====================================================

1. Free Trial

2. Starter Plan
- $20/month
- 1 to 20 users
- 20 AI requests per day per company

3. Growth Plan
- $50/month
- 21 to 50 users
- 75 AI requests per day per company

4. Enterprise Plan
- custom pricing

- custom AI usage limit
