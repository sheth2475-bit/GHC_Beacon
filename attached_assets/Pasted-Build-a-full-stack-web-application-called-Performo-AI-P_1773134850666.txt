Build a full-stack web application called **Performo AI**.

Performo AI is an AI-powered SME performance and execution management platform that combines:
1. KPI Builder
2. PMO / Action Tracker
3. Monthly Review Copilot
4. Dashboard Structure Generator

The goal is to help business owners and managers define the right KPIs, assign actions, review progress, and generate management summaries.

====================================================
1. PRODUCT VISION
====================================================

Create a modern, clean, professional SaaS-style web app for SMEs.

Core promise:
“Performo AI helps SMEs define the right KPIs, track actions, and improve business performance through AI-powered management support.”

Target users:
- Business owners
- General managers
- Department heads
- Operations managers
- HR managers
- Finance managers
- SME leadership teams

Industries supported initially:
- Offshore Helicopter company
- Hospitality

- Retail
- Real Estate

- Maintenance / field services
- Professional services

====================================================
2. CORE FEATURES
====================================================

A. Business Profile Setup
Allow the user to create a company profile with:
- Company name
- Industry
- Company size
- Country
- Departments
- Strategic priorities / goals

B. KPI Generator
User selects:
- Industry
- Department
- Business priorities

Generate KPI recommendations with:
- KPI name
- KPI description
- Formula
- Unit
- Frequency
- Suggested target
- Thresholds: green / amber / red
- KPI owner
- Suggested data source

Departments to support:
- Sales
- Finance
- HR
- Operations
- Procurement
- Customer Service
- Marketing

C. PMO Action Tracker
Allow user to:
- Create meetings
- Add action items
- Assign owner
- Set due date
- Set priority
- Update status
- Mark overdue items
- Filter by department, owner, status, priority

Statuses:
- Not Started
- In Progress
- Completed
- Delayed
- Cancelled

D. Monthly Review Copilot
Allow user to enter monthly KPI actuals and notes.
Then generate an AI-written monthly business review containing:
- Overall performance summary
- KPIs on track
- KPIs below target
- Possible reasons
- Recommended next actions
- Leadership discussion points

E. Dashboard Structure Generator
Generate recommended dashboard layout for the selected business and department.
Include:
- Recommended KPI cards
- Trend charts
- Department summary
- Risk / issue section
- Action tracker section
- Executive summary page structure

F. Basic Analytics Dashboard
Create visual summary widgets for:
- Total KPIs
- KPIs on track
- KPIs below target
- Total actions
- Overdue actions
- Actions completed
- Department performance summary

====================================================

- department
- goals

- kpi_name
- description
- formula
- unit
- frequency
- target_value
- green_threshold

- red_threshold

B. Monthly Review Prompt
The AI should analyze monthly KPI inputs and generate:
- executive summary
- top performing areas
- underperforming areas
- likely causes
- recommended actions
- discussion points for next management meeting

Return structured JSON.

C. Dashboard Planner Prompt
The AI should recommend:
- dashboard pages
- visuals to use
- KPIs to display
- chart types
- why each section matters

Return structured JSON.
